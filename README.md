# Emailer
Emailer is an advanced & premium-quality email sending plugin. It allows you to either connect to the default gmail smtp service or your own, and send real emails to players. Your can either send emails using HTML, or, if you're not familiar with it you can use plain text.

## Prerequisites
- Java 8 or above.
- A spigot server

## Installation
In order to intall and get this plugin to work in your server, you'll first have to buy it from https://www.spigotmc.org/resources/emailer.66184/. You can then find a more documented guide here https://github.com/LagBug/Emailer/wiki/Installation
